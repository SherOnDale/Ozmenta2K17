Technical Workshop Web Development using Flask Micro Framework

Flask is a lightweight micro framework used to develop websites in python based on Werkzeug and Jinja 2 template. Flask has a built-in development server and debugger. Flask is a cross-platform framework with extensions supporting object relational mapper and several common framework related tools.
Prerequisites: Handful experience in HTML, Python, JavaScript
